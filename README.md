# 👩‍🦰 Women Trailer Collection

## 📝 About This Project

Women Trailer Collection is a simple and stylish responsive website built using HTML and CSS.  
It showcases women’s trailers with product images and prices in a clean card layout. The website also includes an order form where users can enter their details to place an order.

This project focuses on creating a modern UI with responsive design that works smoothly on both mobile and desktop devices. It is ideal for beginners who want to practice front-end web development fundamentals.

---

## 💻 Tech Stack

- **HTML5** – For structuring the web pages  
- **CSS3** – For styling, layout, and responsiveness
